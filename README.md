# 04.2 - Array Methods I & II

## Part 1!
1. Create an array that contains 5 different fruits.

2. **Remove the First Element (shift)**
Create a variable. Use the shift method to remove the first element from the fruits array (save this code to the variable, ex: `const removedElement = fruits....()`). Log the modified array and the removed element to the console.

3. **Add a New Element at the Beginning (unshift)**
Use the unshift method to add "Apricot" to the beginning of the fruits array. Log the modified array to the console.

4. **Remove the Last Element (pop)**
Use the pop method to remove the last element from the fruits array. Log the modified array and the removed element to the console.

5. **Add an element to the end (push)**
Use the push method to add an element to the fruits array. Log the modified array to the console.

6. **Using splice to Modify the Array**
Use the splice method to remove "Cherry" and "Date" from the array and add "Fig" and "Grape" in their place. Log the modified array to the console.

## Part 2

You will use advanced array methods to help manipulate some more complicated arrays. We have an array of objects called books.

1. **Using forEach to Log Titles**
Use the forEach method to log each book's title to the console.

2. **Creating a List of Authors with map**
Use the map method to create a new array containing only the authors' names from the books array. Log this new array to the console.

3. **Filtering Books Published Before 1950**
Use the filter method to create a new array containing books that were published before 1950. Log this new array to the console.

4. **Finding a Book by Title with find**
Use the find method to search for a book with the title "1984" and log the found book object to the console.
